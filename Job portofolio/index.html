Honey Encryption Algorithm for Securing Health Records
(Thesis Research Project)

This research shows how this type of encryption can contribute into the field of security and cryptography when it comes to files and data that are vulnerable to brute force attacks. It counter acts the attacker by providing a much more stronger encryption algorithm to prevent such attacks. The only problem is that, given time the attacker may crack the file but it will take them longer to retrieve the file thanks to our developed encryption algorithm.
